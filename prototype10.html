当前题目：电解饱和NaCl溶液

我不理解为什么选项B是错误的？溶液浓度为什么不会减小？

这个问题很好！我们可以从电解原理的角度来分析：

  1. 电解饱和食盐水时，阴极反应是2H₂O + 2e⁻ → H₂↑ + 2OH⁻
  2. 阳极反应是2Cl⁻ → Cl₂↑ + 2e⁻
  3. 总反应为：2NaCl + 2H₂O → 2NaOH + H₂↑ + Cl₂↑

由于Na+和Cl⁻都没有参与反应，但水被消耗了，所以实际浓度会增加而不是减小。

明白了！那为什么pH会增大呢？

这个问题涉及到电解产物：

从阴极反应可以看出，生成了OH⁻离子：

2H₂O + 2e⁻ → H₂↑ + 2OH⁻

所以溶液中OH⁻浓度增加，导致溶液的碱性增强，因此pH值会增大。
