数学 二次函数 难度: 中等

已知二次函数 y = ax² + bx + c 的图像过点 (1, 2), (2, 3), (3, 6)，求 a, b, c 的值。

1245 87

AI解答

解：已知二次函数 y = ax² + bx + c 的图像过点 (1, 2), (2, 3), (3, 6)，将这些点代入函数表达式可得方程组：

a(1)² + b(1) + c = 2
a(2)² + b(2) + c = 3
a(3)² + b(3) + c = 6

化简得：

a + b + c = 2 ...(1)
4a + 2b + c = 3 ...(2)
9a + 3b + c = 6 ...(3)

用消元法解这个方程组：

(2) - (1) 得：3a + b = 1 ...(4)

(3) - (2) 得：5a + b = 3 ...(5)

(5) - (4) 得：2a = 2，所以 a = 1

将 a = 1 代入 (4) 得：3(1) + b = 1，所以 b = -2

将 a = 1, b = -2 代入 (1) 得：1 - 2 + c = 2，所以 c = 3

因此，a = 1, b = -2, c = 3

知识点总结

二次函数 方程组求解 代入法 消元法

数学 2023-06-15

二次函数知识点总结

1 二次函数的定义

二次函数的一般形式为 y = ax² + bx + c，其中 a, b, c 为常数，且 a ≠ 0。二次函数的图像是一条抛物线。

2 二次函数的顶点式

二次函数的顶点式为 y = a(x - h)² + k，其中 (h, k) 是抛物线的顶点坐标。通过配方法可以将一般式转化为顶点式。

y = ax² + bx + c
= a(x² + (b/a)x) + c
= a(x² + (b/a)x + (b/(2a))² - (b/(2a))²) + c
= a(x + b/(2a))² + c - b²/(4a)
其中 h = -b/(2a), k = c - b²/(4a)

3 二次函数的性质

  • 当 a > 0 时，抛物线开口向上；当 a < 0 时，抛物线开口向下。
  • 对称轴为直线 x = -b/(2a)。
  • 顶点坐标为 (-b/(2a), (4ac - b²)/(4a))。
  • 当 a > 0 时，函数在 x = -b/(2a) 处取得最小值；当 a < 0 时，函数在 x = -b/(2a) 处取得最大值。
